{baseurl}/api/authenticate/register

Send Data(Send as Json) : 
{email = email, password = password}

Returns : 
{ Status = false, Error = "Error"}

If error occured "Error" has content, otherwise empty

--------------------

https://{baseurl}/connect/token 

Send Data (FormUrlEncodedContent) : 
["grant_type"] = "password",
["username"] = email,
["password"] = password,
["client_id"] = clientId,
["client_secret"] = client_secret,
["scope"] = "openid offline_access",

Returns : 
{
  "token_type": "Bearer",
  "access_token": "",
  "expires_in": 60,
  "refresh_token": "",
  "id_token": ""
}


--------------------
{baseurl}/connect/token

Send Data :
["grant_type"] = "refresh_token",
["client_id"] = "clientId",
["client_secret"] = "client_secret",
["refresh_token"] = "refresh_token",
["redirect_uri"] = ""

Returns : 
 {
  "scope": "openid offline_access",
  "token_type": "Bearer",
  "access_token": "",
  "expires_in": 60,
  "refresh_token": "",
  "id_token": ""
}
